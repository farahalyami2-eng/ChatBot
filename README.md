# Riyadh Restaurants Chatbot

A web application and REST API for searching restaurants in Riyadh using natural language (Arabic and English).  
The system is built using FastAPI and Pandas, and includes full data preprocessing and logging of all user queries and results for analysis purposes.

## Project Description
This project aims to provide an intelligent search system that allows users to query restaurants in Riyadh using natural language, such as specifying cuisine type, price range, location, or minimum rating.

The system analyzes the user query, applies the appropriate filters to the dataset, and displays the results in a simple chat-based web interface. All interactions are stored in a structured log that can be analyzed later.

## Features
- Comprehensive data preprocessing
- Support for Arabic language queries
- Interactive web-based chat interface
- REST API for restaurant search
- Storage of all user queries and results in a DataFrame
- Export of interaction history to CSV
- Designed for scalability and future extensions
