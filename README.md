# Riyadh Restaurants Chatbot

تطبيق ويب وواجهة برمجية (API) للبحث عن مطاعم الرياض باستخدام اللغة العربية (والإنجليزية)، مبني باستخدام FastAPI و Pandas، مع معالجة مسبقة للبيانات وتخزين جميع الاستفسارات والنتائج لأغراض التحليل.

---

## وصف المشروع
يهدف هذا المشروع إلى توفير نظام بحث ذكي يسمح للمستخدم بالاستعلام عن مطاعم الرياض باستخدام لغة طبيعية، مثل تحديد نوع المطبخ، السعر، الموقع، أو التقييم الأدنى.

يقوم النظام بتحليل الاستفسار، تطبيق فلاتر مناسبة على البيانات، ثم عرض النتائج في واجهة تشات مبسطة، مع تسجيل جميع التفاعلات في سجل قابل للتحليل.

---

## المميزات
- معالجة مسبقة شاملة للبيانات (Preprocessing)
- دعم اللغة العربية في الاستفسارات
- واجهة ويب تفاعلية (Chat UI)
- واجهة REST API للاستعلام
- تخزين جميع الاستفسارات والنتائج في DataFrame
- تصدير سجل التفاعلات إلى ملف CSV
- قابل للتطوير والتوسعة مستقبلًا
